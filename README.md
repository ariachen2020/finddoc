# DocQA - 文件快速問答助手

一個基於 GPT-4 的智能文件問答系統，支持 PDF、Word 和 TXT 文件的智能問答。

## 功能特點

- 支持多種文件格式（PDF、DOCX、TXT）
- 智能文本分塊和向量檢索
- 同義詞匹配和模糊搜索
- 智能答案生成
- 友好的 Web 界面

## 安裝說明

1. 克隆專案：
git clone [您的專案URL]
2. 安裝依賴：

pip install -r requirements.txt

3. 運行應用：
streamlit run app.py


## 使用說明

1. 輸入您的 OpenAI API Key
2. 上傳文件
3. 輸入問題
4. 獲取智能回答

## 注意事項

- 需要有效的 OpenAI API Key
- 建議使用 Python 3.8 或更高版本
- 確保有足夠的硬碟空間用於向量數據庫
